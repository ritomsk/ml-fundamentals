# Titanic Survival Prediction

This repository contains a machine learning project that predicts passenger survival on the Titanic. The project uses data manipulation, exploratory data analysis (EDA), and machine learning models (Decision Tree and Random Forest) to achieve its goal.

## Project Structure

*   **`titanic_survival_prediction.ipynb`**: The main Jupyter Notebook containing the entire pipeline. It includes:
    *   Data loading from `Titanic-Dataset.csv`.
    *   Exploratory Data Analysis (EDA) using `pandas`, `matplotlib`, and `seaborn` to understand feature distributions and relationships with survival.
    *   Data preprocessing:
        *   Handling missing values (e.g., dropping 'Cabin', filling 'Age' based on passenger titles).
        *   Encoding categorical variables ('Sex', 'Embarked', 'Title').
        *   Dropping irrelevant columns ('PassengerId', 'Name', 'Ticket', 'Cabin').
    *   Model training and evaluation:
        *   **Decision Tree Classifier**: Initial model to establish a baseline.
        *   **Random Forest Classifier**: A more robust ensemble model, fine-tuned using `GridSearchCV` to optimize hyperparameters (`n_estimators`, `max_depth`, `criterion`).
    *   The XGBoost Classifier model achieved a best cross-validation score of `0.80` and a test accuracy of `0.816`.
*   **`Titanic-Dataset.csv`**: The primary dataset used for training and testing the models within the notebook. It contains passenger details such as age, sex, ticket class, fare, and whether they survived.
*   **`Submission.csv`**: The final output file containing the survival predictions generated by the trained model.
*   **`data/`**: A directory containing the standard Kaggle dataset files (`train.csv`, `test.csv`, `gender_submission.csv`), which can be used for alternative model evaluations or standard Kaggle submissions.

*   `pandas`
*   `numpy`
*   `matplotlib`
*   `seaborn`
*   `scikit-learn`

```bash
pip install pandas numpy matplotlib seaborn scikit-learn
```

## How to Run

1.  Clone this repository or download the project files.
2.  Ensure you have the required dependencies installed.
3.  Open `titanic_survival_prediction.ipynb` in Jupyter Notebook or JupyterLab.
4.  Run the cells sequentially to observe the data analysis, preprocessing steps, model training, and evaluation results.
